# imerswastorefinalpikselkasir
Tracking & Analytics: Kolom input baru untuk ID Facebook Pixel, TikTok Pixel, dan Google Analytics di menu Pengaturan.

Pajak PPN & Biaya Layanan: Kolom input pengaturan pajak di menu Pengaturan. Di menu Kasir POS, sistem sekarang otomatis ngitung PPN & Biaya Layanan (jika diatur) dan menambahkannya ke Grand Total serta mencetaknya di Struk Thermal.

Role Staf Kasir: * Pengguna (User) biasa sekarang punya akses ke menu Manajemen Pengguna (otomatis berubah nama menjadi "Staf Kasir").

Ada dropdown "Role Akun" saat membuat akun baru (Admin bisa bikin Agency/User/Kasir, User cuma bisa bikin Kasir).

Gue udah nambahin logika keamanan: Kalau yang login itu Kasir, maka menu Katalog, Kategori, Ongkir, dan Pengaturan otomatis disembunyikan. Mereka cuma bisa lihat Dashboard, Pesanan, dan Kasir POS!
